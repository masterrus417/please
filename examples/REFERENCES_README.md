просмотр и редактирование справочников


функция, имитирующая обращение к бэку
1.список справочников (все таблицы, начинающиеся с ref_)
2. по имени справочника возвращающая его поля
3. по имени справочника возвращающая его содержимое
4. по route_id предоставляет список этапов, какие действия, какие акторы могут совершать
* ref_route
* ref_stage
* ref_stage_action_actor
* ref_stage_action_stage
* ref_actor
* ref_action


компонент (страница), который отображает и редактирует все справочники 
* отображение и переход по разным справочникам
* отображение содержимого справочников
* редактирование строк


компонент (страница), которая по разветвленной структуре маршрута предоставляет в графическом виде информацию
* какие этапы следуют друг за другом, какие действия доступны на этих этапах, какие акторы их могут совершать
* обрати внимание, что этапы это не дерево, а направленный граф\
соответсвенно, возможны параллельные несколько этапов, которые могут сойтись в один
